Diccionarios

Los diccionarios sirven para conectar dos conjuntos de datos facilmente, en este caso paises con capitales. Los paises se llaman 'keys' y las capitales 'values'.

xxxxxxxxxx
# Creamos el diccionario
europe = {'spain':'madrid', 'france':'paris', 'germany':'berlin', 'norway':'oslo'}


# Imprimir el diccionario
In [1]: print(europe)
> {'spain': 'madrid', 'france': 'paris', 'germany': 'berlin', 'norway': 'oslo'}


# Imprimir las llaves (paises)
In [2]: print(europe.keys())
> dict_keys(['spain', 'france', 'germany', 'norway'])


# Imprimir la capital de noruega
In [3]: print(europe['norway'])
> oslo

Las llaves no se pueden cambiar, los valores si.

Una misma llave no puede tener mas de un valor.

Un diccionario en este sentido se parece mucho a una funcion matematica.

xxxxxxxxxx
#podemos agregar valores al diccionario asi
europe['england'] = 'london'
#podemos borrar llaves con sus valores asi
del(europe['spain'])

Es posible hacer diccionarios de diccionarios

xxxxxxxxxx
# diccionario en el cual los valores son diccionarios
europe = { 'spain': { 'capital':'madrid', 'population':46.77 },
           'france': { 'capital':'paris', 'population':66.03 },
           'germany': { 'capital':'berlin', 'population':80.62 },
           'norway': { 'capital':'oslo', 'population':5.084 } }


# Para imprimir la capital de francia, accedemos al valor de la llave 'francia',
# el cual es el diccionario {'capital':'paris', 'population':66.03}
# luego accedemos al valor de la llave 'capital', el cual es paris


print(europe['france']['capital'])
> paris


#agregamos italia con sus datos al diccionario
europe['italy'] = {'capital':'rome','population':59.83}


# ahora europa se actualiza agregando italia al final
print(europe)
> { 'spain': { 'capital':'madrid', 'population':46.77 },
    'france': { 'capital':'paris', 'population':66.03 },
    'germany': { 'capital':'berlin', 'population':80.62 },
    'norway': { 'capital':'oslo', 'population':5.084 },
    'italy': {'capital':'rome','population':59.83} }

Pandas

Creado sobre numpy sirve para trabajar con tablas de datos (llamadas dataframes).

Podemos crear tablas de datos a partir de diccionarios, en donde las llaves seran los titulos de las columnas y los valores seran puestos en cada una de esas columnas.

xxxxxxxxxx
import pandas as pd
#creamos diccionario
datos = 
{
    'country': ["Brazil", "Russia", "India", "China", "South Africa"]
    'capital': ["Brasilia", "Moscow", "New Delhi", "Beijing", "Pretoria"]
    'area': [8.516, 17.10, 3.286, 9.597, 1.221]
    'population': [200.4, 143.5, 1252, 1357, 52.98]
}
#convertimos el diccionario a una tablita "dataframe"
brics = pd.DataFrame(datos)
brics.index = ["BR","RU","IN","CH","SA"]
print(brics)

Operadores Booleanos con arreglos numpy

A pesar de que cosas como < o <= funcionan en arreglos de numpy, para usar or, and, not se necesitan otras funciones.

xxxxxxxxxx
import numpy as np
my_house = np.array([18.0, 20.0, 10.75, 9.50])
your_house = np.array([14.0, 24.0, 14.25, 9.0])


# np.logical_and
In [1]: print(np.logical_and(my_house < 11, your_house < 11))
> [False False False True]
